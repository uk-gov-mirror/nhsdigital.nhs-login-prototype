NHS login prototype

This is a prototype that combines the older nhs-login and nhs-cid prototypes and uses the latest NHS design system components.

Much of the complex routing and passing of data has been removed to simplify the code for future designers.

The prototype currently illustrates the NHS App P5 registration journey and then the P9 uplift journey. However it could easily be modified to show other journeys.

Bits missing or incomplete:

  • International phone number entry
  • New OTP page content
  • Take an ID photo
  • Upload an ID photo
